 📱 Advanced Flipcard Quiz Vault – Native Android Application

A high-performance, native Android application engineered in **Java** designed to facilitate interactive self-evaluation via active recall pipelines. The architecture seamlessly integrates a dynamic user quiz interface with an administrative data management terminal.

---

🎯 Project Specifications & Core Features

This application strictly satisfies and implements all core structural project requirements:
* **Interactive Flipcard Mechanics:** Modular quiz cards feature clear prompt placement on the front face, revealing the absolute solution on the back face utilizing an animated **Show/Hide Answer** toggle button.
* **Bi-directional Navigation Layout:** Integrated responsive `Next` and `Previous` boundary navigation handlers to scan back and forth across the active deck seamlessly.
* **Granular Object-Level CRUD System:** Full runtime data control via a dedicated management vault allowing users to dynamically `Add`, `Edit`, and `Delete` technical flashcards on the fly without session disruption.
* **Gamified Scoreboard Matrix:** Tracks active session progress metrics by dynamically calculating and updating the user's score based on evaluation inputs.

---

🛠️ Engineering & Architecture Highlights

* **Synchronized State Engine:** Utilizes a unified memory structure ensuring real-time data persistence and synchronization across separate view contexts without bundle serialization bottlenecks.
* **Fluid 3D Spatial Interpolators:** Implements hardware-accelerated Y-axis rotational transitions via `ObjectAnimator` to mirror authentic physical card material behavior.
* **Array Boundary Safeguards:** Custom pointer tracking protects index evaluation routines during deletion events, fully neutralizing out-of-bounds app crashes.
